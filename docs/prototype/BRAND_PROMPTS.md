# FreelanceFlow — Brand assets finales

## Decisión visual cerrada

La identidad final queda basada en **ledger/document + flow ribbon amber/teal**. Es consistente con `docs/prototype/DESIGN.md`: editorial fintech, seria, clara, freelancer-first y sin estética cripto/trading.

## Regla de uso

- Usar SVG para logos en UI cuando sea posible.
- Usar PNG para favicons, manifest, Open Graph y compatibilidad social.
- No volver a usar imágenes con checkerboard horneado.
- No mezclar con símbolos anteriores tipo nudo/infinito.
- Mantener amber `#F59E0B`, teal `#14B8A6` y navy `#0F172A`.
