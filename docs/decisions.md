# Decisiones vigentes

## Roles de interfaz

FreelanceFlow presenta dos roles:

- **Operativo:** accede a los módulos de gestión y configuración.
- **Administrativo:** tiene acceso restringido a la Bitácora.

## Guardas y persistencia local

La selección de perfil, las guardas de navegación y los datos guardados en el navegador respaldan la experiencia actual de la interfaz. No constituyen autenticación ni autorización reales y no deben interpretarse como controles de seguridad de servidor.

## Límite de plataforma

El producto se entrega actualmente como frontend web estático de múltiples páginas. La persistencia y los datos de ejemplo viven en el navegador; no hay backend, API, procesamiento de pagos ni entrega de comunicaciones externas.

## Fuente de verdad

El código de la aplicación es la fuente de verdad para el comportamiento publicado. La documentación pública describe solo las capacidades y límites que ese comportamiento respalda.
