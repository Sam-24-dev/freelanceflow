# Historias de usuario

Estas historias describen el comportamiento actual del frontend.

## Visitante público

- **US-PUB-01:** Como visitante público, quiero conocer el propósito de FreelanceFlow para decidir si responde a mis necesidades de gestión.
- **US-PUB-02:** Como visitante público, quiero elegir un perfil de trabajo para entrar a la interfaz correspondiente.

## Operativo

- **US-OP-01:** Como Operativo, quiero consultar un dashboard de ingresos, gastos, flujo neto y cuentas por cobrar para revisar mi situación financiera.
- **US-OP-02:** Como Operativo, quiero registrar y filtrar movimientos para ordenar ingresos y gastos.
- **US-OP-03:** Como Operativo, quiero gestionar clientes y su contacto principal para mantener mi directorio comercial.
- **US-OP-04:** Como Operativo, quiero crear y mantener proyectos para seguir su avance, tiempo y rentabilidad estimada.
- **US-OP-05:** Como Operativo, quiero definir servicios reutilizables para incorporarlos a mi trabajo comercial.
- **US-OP-06:** Como Operativo, quiero crear propuestas, actualizarlas y marcar su resultado para organizar oportunidades.
- **US-OP-07:** Como Operativo, quiero convertir una propuesta aceptada en proyecto para continuar el trabajo acordado.
- **US-OP-08:** Como Operativo, quiero gestionar facturas y registrar pagos para seguir saldos y vencimientos.
- **US-OP-09:** Como Operativo, quiero consultar reportes y exportar resultados disponibles para revisar la operación.
- **US-OP-10:** Como Operativo, quiero configurar categorías, parámetros fiscales estimados y preferencias de facturación para adaptar la interfaz a mi trabajo.
- **US-OP-11:** Como Operativo, quiero revisar alertas y ajustar sus preferencias visibles para mantener atención sobre eventos financieros.
- **US-OP-12:** Como Operativo, quiero actualizar los datos básicos de mi cuenta para personalizar mi perfil de trabajo.

## Administrativo

- **US-ADM-01:** Como Administrativo, quiero consultar la Bitácora para revisar la actividad operativa registrada.
- **US-ADM-02:** Como Administrativo, quiero limpiar los registros de la Bitácora en el navegador para reiniciar la vista de actividad local.
